## ⚙️ How to Run pc-detox

pc-detox uses the Windows command line to run commands. Here’s how to run the tool:

1. Press the **Windows key** on your keyboard or click the **Start menu** button.

2. Type `cmd` in the search bar.

3. Click **Command Prompt** from the search results to open it.

4. In the Command Prompt window, type the command below and press **Enter**:

   ```
   npx pc-detox
   ```

5. pc-detox will start running. You will see messages showing what it is doing.

6. The process may take a few minutes depending on your PC.

7. When finished, pc-detox will show a summary of the cleanup.

## 💡 What Does pc-detox Do?

pc-detox works behind the scenes to improve your PC. It does these main tasks:

- **Remove Bloatware**: Deletes apps that come pre-installed but rarely used.
- **Clean Startup Programs**: Stops unnecessary programs from running when Windows starts.
- **Clean Registry**: Fixes errors in the Windows registry to improve stability.
- **Clear Temporary Files**: Deletes files that slow down your disk.
- **Monitor Performance**: Checks CPU and memory use during cleanup.
- **User Friendly Logs**: Shows easy-to-understand results at the end.

The tool runs safely and only removes files and apps that are not needed for normal use.

## 📋 Common Terms

- **Bloatware**: Software included with your PC that you do not need.
- **Registry**: A database where Windows stores system settings.
- **Startup programs**: Apps that start automatically when you turn on your PC.
- **Command Prompt**: A text window to enter commands on Windows.
- **Node.js**: A tool pc-detox uses to run on your PC.
- **Temporary files**: Files created during normal use that can be safely deleted.
